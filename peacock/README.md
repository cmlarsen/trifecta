#Peacock
Front-end web application for Trifecta

## How to build

### [Web Application build instructions](peacock-web/README.md)

### [Server build instructions](peacock-server/README.md)

## How to run

1. Web application

  ```
  cd peacock-web
  gulp
  ```
2. Server

  ```
  sbt
  > project peacock-server
  > run
  ```
